Who am I?

I'm a cybersecurity researcher and agentic AI developer who builds tools that autonomously hunt, exploit, and patch software vulnerabilities. I don't just run scanners — I write the systems that do.

My flagship project is Mutagen — an autonomous zero-day fuzzer that disassembles C binaries with Ghidra Headless, catches heap overflows and buffer corruptions across 20+ CVE targets, generates working exploit PoCs, and patches the source code in a closed loop. No human in the loop required.

On the offensive side, I work with Nmap, Metasploit, Burp Suite, and John the Ripper across real CTF environments on TryHackMe (91 badges earned). On the AI side, I fine-tune large language models on Hugging Face for automated vulnerability classification and exploit verification.

How Mutagen Works

End-to-end autonomous vulnerability pipeline — no human in the loop.

📥
1. Ingest Target binary + source code fed into the pipeline
🔬
2. Disassemble Ghidra Headless decompiles binaries into analyzable C
💥
3. Fuzz & Exploit AI-guided fuzzing catches crashes, generates working PoCs
🩹
4. Patch Gemini API generates and applies source-level patches
✅
5. Verify Re-runs fuzz cycles — 0 crashes confirms target is secure
