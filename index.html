I am a cybersecurity researcher and agentic AI developer dedicated to automating offensive and defensive operations. I build pipelines that crawl codebase repositories, catch buffer corruptions, evaluate system vulnerabilities, and implement dynamic updates autonomously.

My core project is Mutagen, an autonomous zero-day fuzzer that decompiles targets via Ghidra, analyzes memory limits, writes working exploit proof-of-concepts, and automatically updates safety checks to secure C codebases.

I fine-tune large language models on Hugging Face to classify vulnerabilities and match MITRE ATT&CK maps, utilizing isolated VM environments for exploit verification.
